# MyMeds

MyMeds is a medicine management and reminder web application that helps users track medications, manage dosage schedules, monitor medicine inventory, and improve medication adherence through reminders and progress tracking.

## Features

- Medicine scheduling and dosage tracking
- Browser-based medication reminders
- User authentication and session management
- Medicine inventory and stock monitoring
- Low-stock alert system
- Daily medication status tracking
- Adherence streak visualization
- Dynamic filtering and category-based organization
- Persistent local data storage using LocalStorage
- Responsive and user-friendly interface

## Technology Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Browser Notifications API
- LocalStorage API

## Project Overview

Managing medications consistently can be challenging, especially for individuals who need to take multiple medicines throughout the day. MyMeds simplifies medication management by providing an organized system for scheduling medicines, setting reminders, tracking adherence, and monitoring medicine inventory.

The application allows users to create personalized medication schedules, receive browser notifications, monitor medicine stock levels, and maintain a history of medication adherence through an intuitive dashboard.

## Key Functionalities

### Medicine Management
- Add, edit, and organize medications
- Categorize medicines based on type
- Configure dosage schedules and medicine timings
- Maintain medicine inventory records

### Reminder System
- Create and manage medication alarms
- Browser notification support
- Custom reminder labels
- Enable or disable alarms as needed

### Adherence Tracking
- Mark medicines as Taken, Skipped, or Pending
- View medication history
- Monitor adherence streaks
- Track daily progress

### Inventory Monitoring
- Real-time stock tracking
- Automatic stock reduction after dosage completion
- Low-stock alerts
- Medicine consumption monitoring
